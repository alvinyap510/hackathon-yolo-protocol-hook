## How To Run

### 1. Git clone the repo
```
git clone git@github.com:alvinyap510/hackathon-yolo-protocol-hook.git ${YOUR_FOLDER_NAME}
```

### 2. Install node modules
```
cd ${YOUR_FOLDER_NAME}
pnpm install
```

### 3. Install/upgrade Foundry & Forge deps
```
foundryup
forge install

```

### 4. Run the tests
```
forge test
```
